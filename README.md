# BST
Important Points:
1. No Duplicates
2. Inorder traversal gives sorted order
3. If there are "n" nodes present then no. of binary search trees that can be produced is:
          (2nCn)/(n+1)
For Example:
if n=3, then (2*3)C3/(3+1)
              6C3/4
              (6!/3!6!)/4
         --------Ans=5-------
